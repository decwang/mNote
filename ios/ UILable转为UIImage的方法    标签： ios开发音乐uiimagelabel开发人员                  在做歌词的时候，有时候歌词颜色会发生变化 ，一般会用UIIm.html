UILable转为UIImage的方法

标签： ios开发音乐uiimagelabel开发人员

         在做歌词的时候，有时候歌词颜色会发生变化 ，一般会用UIImage做剪裁，慢慢出现不同的颜色，用UIlable专为UIImage，图片进行绘图操作就可以达到想要的效果了

  label转化成image：

[objc] view plain copy
  1. - (UIImage*) imageWithUIView:(UIView*) view  
  2. {  
  3.     UIGraphicsBeginImageContext(view.bounds.size);  
  4.     CGContextRef ctx = UIGraphicsGetCurrentContext();  
  5.     [view.layer renderInContext:ctx];  
  6.     UIImage* tImage = UIGraphicsGetImageFromCurrentImageContext();  
  7.     UIGraphicsEndImageContext();  
  8.     return tImage;  
  9. }  

然后进行绘图剪裁

[objc] view plain copy
  1. - (UIImage *)croppedImage:(UIImage *)image cutRect:(NSInteger)width  
  2. {  
  3.     if (image)  
  4.     {  
  5.         float min = MIN(image.size.width,image.size.height);  
  6.         CGRect rectMAX = CGRectMake(0, 0, width, min);  
  7.         CGImageRef subImageRef = CGImageCreateWithImageInRect(image.CGImage, rectMAX);  
  8.         UIGraphicsBeginImageContext(rectMAX.size);  
  9.         CGContextRef context = UIGraphicsGetCurrentContext();  
  10.         CGContextDrawImage(context, CGRectMake(0, 0, width, min), subImageRef);  
  11.         UIImage *viewImage = [UIImage imageWithCGImage:subImageRef];  
  12.         UIGraphicsEndImageContext();  
  13.         CGImageRelease(subImageRef);  
  14.         return viewImage;  
  15.     }  
  16.       
  17.     return nil;  
  18.       
  19. }  

这样就可以根据时间出现缓慢的出现歌词了。。。。。。

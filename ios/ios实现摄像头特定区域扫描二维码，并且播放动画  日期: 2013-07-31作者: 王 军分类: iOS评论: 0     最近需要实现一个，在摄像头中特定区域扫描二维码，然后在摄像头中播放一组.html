ios实现摄像头特定区域扫描二维码，并且播放动画

日期: 2013-07-31作者: 王 军分类: iOS评论: 0 

最近需要实现一个，在摄像头中特定区域扫描二维码，然后在摄像头中播放一组动画的需求，其中播放动画不难，主要是在摄像头特定区域扫描二维码，经过研究，使用Zxing可以做到这一点，下面是实现效果和过程

源代码：https://github.com/marcuswang/TestZxing

第一，项目配置zxing

我这里面就不说怎么配置了，可以参考这篇博客：http://blog.devtang.com/blog/2012/12/23/use-zxing-library/

5     if (encode) {

8     CVImageBufferRef imageBuffer = CMSampleBufferGetImageBuffer(sampleBuffer);

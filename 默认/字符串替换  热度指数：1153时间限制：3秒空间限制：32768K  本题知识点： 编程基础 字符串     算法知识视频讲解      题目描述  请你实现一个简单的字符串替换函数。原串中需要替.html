题目描述

请你实现一个简单的字符串替换函数。原串中需要替换的占位符为"%s",请按照参数列表的顺序一一替换占位符。若参数列表的字符数大于占位符个数。则将剩下的参数字符添加到字符串的结尾。

给定一个字符串A，同时给定它的长度n及参数字符数组arg，请返回替换后的字符串。保证参数个数大于等于占位符个数。保证原串由大小写英文字母组成，同时长度小于等于500。

测试样例：
"A%sC%sE",7,['B','D','F']
返回："ABCDEF"

  3. public class StringFormat {  
  4.     public String formatString(String A, int n, char[] arg, int m) {  
  5.         int i=0;  
  6.         StringBuffer sb = new StringBuffer(A);  
  7.         while(sb.indexOf("%s")!=-1){  
  8.             int k=sb.indexOf("%s");  
  9.             sb.replace(k,k+2,String.valueOf(arg[i++]));  
  10.         }  
  11.         while(i<m)sb.append(arg[i++]);  
  12.         return sb.toString();  
  13.     }  
  14. }  

注意： 

1.如果使用String 的replaceFirst或者 replace 方法时候，一定要重新定义新的字符串，因为String是final的，本人不会改变。

2.而使用StringBuffer会更方便，可以直接改变本身。

3.StringBuffer没有replaceFirst方法，只有replace方法。

public StringBuffer replace(int start, int end, String str)

参数

  • start -- 这是开始索引(包括)。

  • end -- 这就是结束索引(不包括)。

  • str -- 这是字符串，它将取代以前的内容。
